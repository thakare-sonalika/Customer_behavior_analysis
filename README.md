# 📊 Data Analysis Project – End-to-End Business Insights

## 🔎 Overview

This project demonstrates an end-to-end data analysis workflow — from raw dataset to actionable business insights.

It covers:

* Data loading and preprocessing using Python
* Exploratory Data Analysis (EDA)
* Data cleaning and transformation
* SQL querying using PostgreSQL / MySQL / SQL Server
* Interactive dashboard development in Power BI
* Report creation and presentation (PPT via Gamma)

The objective of this project is to transform raw data into meaningful insights that support data-driven decision-making.

---

## 📁 Dataset

The dataset contains structured transactional/business data including:

* Customer information
* Product details
* Sales transactions
* Revenue and quantity metrics
* Time-based variables (Date, Month, Year)

The dataset was imported in Python for cleaning and exploration, and later stored in a relational database for SQL analysis.

## ⚙️ Project Workflow / Steps

### 1️⃣ Data Loading (Python)

* Imported dataset using Pandas
* Checked structure, data types, and summary statistics
* Identified missing values and inconsistencies

### 2️⃣ Data Cleaning

* Handled missing/null values
* Removed duplicates
* Standardized column names
* Converted data types (dates, numeric fields)
* Created derived features (e.g., Month, Revenue)

### 3️⃣ Exploratory Data Analysis (EDA)

* Analyzed sales trends over time
* Identified top-performing products
* Examined customer purchasing patterns
* Visualized revenue distribution and category performance

### 4️⃣ SQL Analysis

* Loaded cleaned dataset into PostgreSQL / MySQL / SQL Server
* Performed queries such as:

  * Revenue by category
  * Top purchased products
  * Customer segmentation
  * Repeat customer analysis
  * Aggregations using GROUP BY, JOIN, HAVING

### 5️⃣ Power BI Dashboard

* Created interactive dashboard with:

  * KPI Cards (Total Sales, Profit, Quantity)
  * Time-series sales trends
  * Category-wise performance
  * Customer segmentation
  * Filters/Slicers for dynamic analysis

 6️⃣ Report & Presentation

* Summarized key insights in a structured business report
* Designed a professional presentation using Gamma
* Highlighted data-driven recommendations
